## Datasets

### 1. Movielens

- Download the dataset from: https://grouplens.org/datasets/movielens/latest/

- Download: ml-latest-small.zip

- Extract it to the following directory: data/ml-latest-small/

### 2. The Movies Dataset

- Download from Kaggle: https://www.kaggle.com/datasets/rounakbanik/the-movies-dataset

- Extract selected files to: data/archive/

Directory structure:

- data/
    - ml-latest-small/
    - archive/
